# Python基础-注释

## 1. **单行注释**

用 `#` 开头，后面跟注释内容。

```python
# 这是一个单行注释
name = "Alice"  # 给变量 name 赋值
```

> `#` 后面最好加一个空格（PEP8 风格）

---

## 2. **多行注释**（多行 `#`）

Python 没有真正的多行注释语法，但我们可以用连续的 `#` 来模拟：

```python
# 这个函数是用来计算
# 两个数的最大公约数
# 使用的是辗转相除法
def gcd(a, b):
    while b:
        a, b = b, a % b
    return a
```

---

## 3. **文档字符串（Docstring 注释）**

用三引号 `""" """` 或 `''' '''` 括起来，常用于：

- 函数、类、模块的说明
- 可以被 `help()` 函数调用到

```python
def add(x, y):
    """
    这是一个求和函数
    参数:
        x: 第一个加数
        y: 第二个加数
    返回:
        两数之和
    """
    return x + y
```
